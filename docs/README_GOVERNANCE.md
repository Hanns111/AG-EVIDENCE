# GOVERNANZA DEL PROYECTO AG-EVIDENCE

Este directorio contiene los archivos que gobiernan
la arquitectura, el contexto, la continuidad y la toma de decisiones
del proyecto AG-EVIDENCE.

## Regla Absoluta
Ninguna IA puede generar código, proponer cambios
o refactorizar sin haber leído estos archivos.

## Clasificación
Archivos ESTABLES:
- PROJECT_SPEC.md
- HARDWARE_CONTEXT.md
- ARCHITECTURE.md
- GOVERNANCE_RULES.md
- ADR.md

Archivos VIVOS:
- CURRENT_STATE.md
- CONTEXT_CHAIN.md

## Uso Correcto
- CURRENT_STATE.md se actualiza al final de cada sesión
- ADR.md solo se modifica creando nuevas entradas
- CONTEXT_CHAIN.md se usa cuando se cambia de chat o IA

Estos archivos son la **fuente de verdad del proyecto**.
