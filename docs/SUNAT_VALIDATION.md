# VALIDACIÓN SUNAT — DISEÑO

## Objetivo

Permitir corroboración externa de comprobantes procesados por el sistema.

## Problema

Los datos extraídos pueden ser correctos estructuralmente, pero requieren validación oficial.

## Solución

Integración automatizada con SUNAT simulando comportamiento humano.

## Restricciones

- Evitar detección como bot
- No usar Selenium básico
- No usar APIs pagas

## Tecnologías a evaluar

- Playwright (stealth)
- Análisis de endpoints usados por SUNAT
